# SaaS em Oracle SQL

Projeto desenvolvido com foco em prática de modelagem de banco de dados e manipulação de dados utilizando Oracle SQL. O objetivo principal é demonstrar conhecimentos básicos em estruturação de banco, consultas e organização de dados em um cenário próximo ao real.

## Tecnologias utilizadas

* Oracle SQL
* PL/SQL

## Funcionalidades

* Criação de relatórios personalizados
* Relacionamentos entre entidades
* Consultas SQL (simples e avançadas)
* Procedures e packages básicas

## Objetivo

Este projeto foi desenvolvido como parte de estudos e prática pessoal, com foco em consolidar fundamentos de banco de dados.

## Observações

* Projeto sem fins comerciais
* Estrutura simplificada
* Pode conter melhorias futuras
